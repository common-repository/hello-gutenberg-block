== Description ==
In tribute to the famous "Hello Gutenberg Block" plugin by Hitendra Chopda comes this new plugin. 
And how could someone possible name a new default plugin other than "Hello Gutenberg Block", as it's THE definition for a default block example :)

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or higher

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
